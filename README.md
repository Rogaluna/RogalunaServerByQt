# RogalunaServerByQt

提供给 [rogaluna-web](https://github.com/Rogaluna/rogaluna-web) 的服务程序，使用 mingw 编译，支持 Windows 和 Linux 平台。需要配置Postgre数据库，可在运行后编辑自动生成的db_config.ini文件进行配置。

未提供

## 模块列表

***QJsonWebToken***  jwt验证生成器
***QTagLib***  taglib的Qt兼容库，提供音频元数据的获取和编辑
